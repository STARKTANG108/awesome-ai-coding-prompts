# Contributing Guide

Thank you for considering contributing to this project!

## How to Contribute

1. Fork this repository
2. Add your prompt template to the appropriate category
3. Make sure your prompt is:
   - Clear and reusable
   - Tested with at least one AI tool
   - Includes a brief description of use case
4. Submit a Pull Request

## Prompt Template Format

Each prompt should follow this structure:

### [Prompt Title]
[One-line description of what this prompt does]

[Prompt content with [placeholders] in brackets]

## Questions?

Open an issue or start a discussion!
